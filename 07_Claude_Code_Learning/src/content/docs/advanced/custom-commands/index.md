## カスタムコマンドとは

カスタムコマンドは、Claude Codeで頻繁に行うタスクを自動化・再利用可能にする強力な機能です。

単純なテキスト置換から、複雑なマルチステージワークフローまで、あらゆるタスクをカプセル化できます。

## 基本構造

カスタムコマンドはMarkdownファイルとして作成します：

```markdown
---
description: コマンドの簡潔な説明
---

# コマンド名

実行するタスクの詳細な説明...

## ステップ1

最初のステップの内容...

## ステップ2

次のステップの内容...
```

## 配置場所

カスタムコマンドは以下の場所に配置します：

```
.claude/commands/
├── my-command.md
├── another-command.md
└── category/
    └── specialized-command.md
```

## パラメータの使用

カスタムコマンドでパラメータを使用する方法：

```markdown
---
description: ファイルを処理するコマンド
---

# ファイル処理

`{{ file_path }}` パラメータで指定されたファイルを処理します。
```

## コマンドチェーン

複数のコマンドを連結して複雑なワークフローを構築：

```markdown
---
description: 複合ワークフロー
---

# ワークフロー実行

1. 最初のコマンドを実行
2. 結果を検証
3. 次のコマンドを実行
```

## ベストプラクティス

1. **明確な説明**: `description`でコマンドの目的を簡潔に説明
2. **ステップの分割**: 複雑なタスクは小さなステップに分割
3. **ドキュメント**: コマンド内で詳細な説明を提供
4. **再利用性**: 汎用的なコマンドを作成
