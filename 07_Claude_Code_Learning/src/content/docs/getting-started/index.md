## このセクションについて

このセクションでは、Claude Codeの基本的なセットアップと使い方を説明します。

既にClaude Codeをインストールして使用している方は、[基本概念](/core-concepts/)セクションへ進むことができます。

## 前提条件

- Node.js 18.x以上がインストールされていること
- Gitの基本的な知識
- ターミナル操作の基本知識

## インストールと設定

Claude Codeのインストール方法は[公式ドキュメント](https://docs.anthropic.com/en/docs/claude-code)を参照してください。

基本的なインストールコマンド：

```bash
npm install -g @anthropic-ai/claude-code
```
