# Research Document Repository

**最終更新**: 2025-12-15

## 📁 リポジトリ概要

このリポジトリは、AIビジネスコンサルティング、ニュース分析、個人研究プロジェクトのドキュメントを管理するための中央集約型ストレージです。

## 🗂️ フォルダ構成

### 📁 01_Business/
ビジネス関連のドキュメント
- `AI_Side_Business/`: AI副業事業の計画書、実行記録、市場調査

### 📁 02_News/
ニュースレポートと分析
- `YYYY/MM/`: 年月別整理
- `AI_Technology/`: AI技術ニュース
- `Domestic/`: 国内ニュース
- `International/`: 国際ニュース

### 📁 03_Personal/
個人的コンテンツ
- `Entertainment/`: エンターテイメント関連
- `TV_Drama/`: テレビドラマの考察分析

### 📁 04_Research/
調査・研究プロジェクト
- `AI_Tools/`: AIツールの調査と比較
- `Industry_Analysis/`: 業界分析レポート

### 📁 05_Archive/
古いドキュメントのアーカイブ

### 📁 06_Templates/
ドキュメントテンプレート

### 📁 99_System/
システム管理ファイル

## 📝 命名規則

基本フォーマット: `YYYY-MM-DD_category_description.md`

### カテゴリコード
- `business`: ビジネス関連
- `research`: 調査・分析
- `news`: ニュース・レポート
- `personal`: 個人的コンテンツ
- `template`: テンプレート
- `archive`: アーカイブ

## 🔧 管理ルール

1. **新しいファイル**: 必ず対応フォルダに保存
2. **命名規則**: 統一されたフォーマットを使用
3. **バージョン管理**: 重要な変更はGitで管理
4. **定期整理**: 月次で不要なファイルをアーカイブへ移動

## 📞 お問い合わせ

不明点や改善提案は、リポジトリ管理者までご連絡ください。
