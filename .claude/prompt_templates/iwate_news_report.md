# 岩手県最新ニュースレポート作成プロンプト

## 基本指示
現在時刻から24時間以内に発生した岩手県の最新ニュースを収集し、詳細な分析レポートを作成してください。

## 検索フェーズ
### 1. 主要検索の実施
以下のクエリでmcp__web-search-prime__webSearchPrimeを実行：
- `岩手県 ニュース 今日 最新`（recency: oneDay, content_size: high）
- `岩手県 事故 災害 ニュース`（recency: oneDay, content_size: high）
- `岩手県 公式発表 ニュース`（recency: oneDay, content_size: high）

### 2. 重要度判定
検索結果から以下の優先順位でニュースを選択：

#### ★★★★★ 緊急事態
- 人身被害（死亡・負傷）
- 大規模災害・事故
- 警報・注意報
- 重要インフラ障害

#### ★★★★☆ 重要事象
- 県政発表・重要施策
- 大規模イベント
- 施設運営の変更

#### ★★★☆☆ 地域動向
- 教育・医療・福祉
- 経済・雇用・観光

## 詳細分析フェーズ
### 各ニュースの分析内容
1. **基本情報**
   - 発生時刻（可能な限り正確に）
   - 具体的な場所
   - 関係者（当事者）
   - 被害状況

2. **深掘り分析**
   - 背景要因（なぜ起きたのか）
   - 統計データ（過去との比較）
   - 関連事例
   - 専門家の見解

3. **影響評価**
   - 地域への直接的影響
   - 経済的影響
   - 社会的影響
   - 今後の見通し

4. **対応策**
   - 行政の対応
   - 住民への注意点
   - 予防策

## 品質管理ルール
- 100%事実ベース（推測を排除）
- 信頼できる情報源のみ使用
- 具体的な数値・データを必ず含める
- 複数の情報源で確認可能な情報のみ使用
- 誇情的な表現を避け、客観的な記述に徹する
