# 岩手県最新ニュースレポート作成コマンド

## 概要
2025年12月5日以降、岩手県の24時間以内の最新ニュースを収集し、詳細な分析レポートを作成するコマンド。

## 使用方法
```
/latest-iwate-news
```

## ワークフロー

### 1. 情報収集フェーズ
- WebSearchPrimeで「岩手県 ニュース 今日 最新」を検索
- 検索期間: oneDay（24時間以内）
- コンテンツサイズ: high（詳細情報）

### 2. 主要ニュースの優先順位
1. **緊急事態**（★★★★★）
   - 事故・災害、人身被害
   - 緊急対応、警報・注意報

2. **重要事象**（★★★★☆）
   - 県政発表、重要施策
   - 大規模イベント、施設運営

3. **地域動向**（★★★☆☆）
   - 教育・医療・福祉
   - 経済・雇用・観光

### 3. 詳細分析の構成要素
- **基本情報**: 発生時刻、場所、当事者
- **状況詳細**: 現場状況、被害規模、対応状況
- **背景分析**: 原因、統計データ、年度比較
- **影響評価**: 地域への影響、今後の見通し
- **対策情報**: 行政対応、住民への注意点

### 4. 執筆フォーマット
```markdown

*現在時刻の24時間以内に発生した主要ニュースを詳細に分析*

## 🔴 **【重要度：★★★★★】[タイトル]
**発生時刻**:

### 詳細分析
[内容]

### 背景と影響
[分析]

```

### 5. 情報源の明記
- 一次情報源（公式発表）
- 報道機関（地元メディア）
- 統計データ（政府・自治体）

## 品質基準
- 100%事実ベース（推測・予測を排除）
- 信頼できる情報源のみ使用
- 具体的な数値・データを含める
- 深掘り分析で付加価値提供
